## About Umka Player

Umka Player is a multimedia software — the reference implementation of the Umka Kiosk Standard, an open protocol for museum kiosk systems. The Umka Kiosk Standard and the Smart Museum "Umka" system are developed and owned by Multimedia Solutions Lab Ltd.

The Umka Kiosk Standard v1.26.2 defines:

- MQTT protocol for museum kiosk remote control
- CMS-agnostic REST API for content synchronization
- Operating modes
- Content package format and structure
- Local-first content architecture
- Guide tablet integration
- IoT device integration

All implementations of the Umka Kiosk Standard will naturally contain similar
code because they implement the same protocol specification. This is by design
and is the intended purpose of having an open standard.

Client-specific customizations (branding, museum-specific features, custom
content types) built on top of this reference implementation remain proprietary
to the respective clients.
